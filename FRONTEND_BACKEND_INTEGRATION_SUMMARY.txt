🔌 API INTEGRATIONS
═════════════════════════════════════════════════════════════════════════════

EXTRACTION (File Upload/Text Paste)
  Endpoint: POST /api/extract
  Input:    { "text": "raw complaint text" }
  Output:   { "success": true, "extracted_data": {...} }
  Frontend: useFileUpload hook → AIAssistantPanel
  Flow:     Upload → Validate → Extract → Populate Form

SAVE COMPLAINT
  Endpoint: POST /api/complaints
  Input:    { complaint form data }
  Output:   { "id": 42, "created_at": "...", ... }
  Frontend: ComplaintIntakeForm → Save button
  Flow:     Validate → Save → Show ID → Reset Form

AI CHAT
  Endpoint: POST /api/chat
  Input:    { "complaint_text": "...", "question": "..." }
  Output:   { "answer": "AI response...", "success": true }
  Frontend: ChatComponent → Chat input
  Flow:     Ask Question → Get Complaint Context → Call API → Display Response

RISK ANALYSIS (Bonus)
  Endpoint: POST /api/analyze-risk
  Input:    { complaint form data }
  Output:   {
    "completeness_score": 85,
    "root_cause_suggestion": "...",
    "capa_recommendation": "...",
    "duplicate_flag": false,
    "risk_level": "High"
  }
  Frontend: ComplaintIntakeForm → "Analyze Risk" button
  Flow:     Validate → Analyze → Display Results in Panel

🚀 DATA FLOW PIPELINES
═════════════════════════════════════════════════════════════════════════════

EXTRACTION PIPELINE:
  User Action → File/Text Input
    ↓
  Validation (type, size)
    ↓
  Extract text content
    ↓
  Redux: uploadStart, uploadProgress
    ↓
  API: POST /api/extract
    ↓
  Backend: LangGraph extraction
    ↓
  Response: Extracted JSON
    ↓
  Redux: populateFormFromExtraction
    ↓
  UI: Form auto-populates, progress completes

SAVE PIPELINE:
  User Action → Click "Save Complaint"
    ↓
  Validate form (all required fields)
    ↓
  Redux: saveComplaintStart
    ↓
  API: POST /api/complaints
    ↓
  Backend: Save to database
    ↓
  Response: Complaint ID
    ↓
  Redux: saveComplaintSuccess
    ↓
  UI: Success message, form resets

CHAT PIPELINE:
  User Action → Type message, click Send
    ↓
  Display user message immediately
    ↓
  Get complaint context from Redux
    ↓
  Redux: setLoading(true)
    ↓
  API: POST /api/chat
    ↓
  Backend: Call Groq API
    ↓
  Groq: Process with Gemma2-9b-it
    ↓
  Response: AI analysis
    ↓
  Display AI response with timestamp
    ↓
  Redux: setLoading(false)

RISK ANALYSIS PIPELINE:
  User Action → Click "Analyze Risk"
    ↓
  Validate form fields
    ↓
  Redux: analyzeRisk(formData)
    ↓
  API: POST /api/analyze-risk
    ↓
  Backend: LangGraph analysis
    ↓
  Groq: Analyze completeness, risks, duplicates
    ↓
  Response: Scores and recommendations
    ↓
  Display: RiskAnalysisPanel with results

🔄 REDUX STATE MANAGEMENT
═════════════════════════════════════════════════════════════════════════════

COMPLAINT SLICE:
  State:
    form: ComplaintFormData
    isLoading: boolean
    error: string | null

  Actions:
    updateFormField
    populateFormFromExtraction
    resetForm
    saveComplaintStart/Success/Error

EXTRACTION SLICE:
  State:
    isExtracting: boolean
    progress: number (0-100)
    status: 'idle'|'uploading'|'extracting'|'completed'|'error'
    error: string | null
    extractedText: string | null
    uploadedFileName: string | null

  Actions:
    uploadStart, uploadProgress
    extractionStart, extractionProgress
    extractionSuccess, extractionError
    resetExtraction

CHAT SLICE:
  State:
    messages: ChatMessage[]
    isLoading: boolean
    error: string | null

  Actions:
    addMessage
    setLoading, setError
    clearChat
